## Getting Started

### Installation

1. Clone the repo

   ```sh
    git clone https://github.com/eldadYikne/gambling-front.git

   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. Open the terminal and enter the following line

   ```js
   npm run dev
   ```
